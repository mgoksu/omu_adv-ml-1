# İleri Makine Öğrenme - Ödev 1


Model oluşturmak için train scriptini çalıştırın.
```
python train.py
```
Eğitim bittiğinde `model.pt` dosyası oluşacaktır.

Devamında oynatıp izlemek için play scriptini çalıştırın.
```
python play.py
```
